# employee-attrition-dashboard
An interactive and insightful Power BI dashboard designed to analyze employee attrition trends across various dimensions. The dashboard provides a comprehensive view of factors contributing to employee turnover, including:

- Department
- Salary Level
- Job Role
- Education Background
- Tenure (Length of Service)
